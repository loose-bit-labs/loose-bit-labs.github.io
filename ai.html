Face Mesh Workflow

This project combines mediapip with Zoe and Midas depth. It identifies a primary face and provides knobs to tune each depth source for the face mesh. Then you can combine then linearly to create a composite which better reflects the ground truth. Since the (expensive) depth estimation is done only once there is a large degree of freedom to configure the output and tweak the final result.
